# Changelog

Заметные изменения проекта. Раздел ниже описывает локальные коммиты, ещё не
отправленные в `origin/master`.

## [Unreleased]

### Добавлено

- Экран «История списаний» (`src/popup/views/HistoryPage.vue`) по всем задачам
  текущего пользователя с группировкой по дате и удалением записей. Данные
  берутся из глобального эндпоинта `/api/workItems?author=me`; добавлен метод
  `getAllWorkItems` в контракт сообщений, воркер и фасад.
- Выбор инстанса YouTrack на экране логина: поле ввода URL заменено на список
  зашитых в код инстансов (`src/popup/instances.ts`).
- Подтверждение удаления токена — диалог «Вы уверены?» (Да/Нет) на экране
  настроек.
- Контекстная навигация в шапке: стрелка «назад» на экранах настроек и истории,
  иконки настроек и истории на главном экране.
- Иконки расширения (16/48/128) и favicon; центрированная dev-рамка 600x600 для
  режима `npm run dev`.

### Изменено

- Переход между экранами: слайд-анимация заменена на плавное затухание 0.15s.
- Шапка модалок (`TaskModalHeader`) центрируется по горизонтали и вертикали.
- Добавлены hover-состояния на кликабельные элементы интерфейса.
- Списки на экранах `/` и `/history` перерисованы карточками: белые карточки на
  светлом фоне, чип с ID в фирменном фиолетовом оттенке, акцент на времени,
  hover с подсветкой и тенью. В истории заголовки дат стали «липкими»
  (uppercase), тип работы вынесен в пилюлю.
- Модалка списаний по задаче (`TaskWorkItemsModal`) приведена к тому же
  карточному стилю: светлый фон тела, чип с датой, пилюля типа, перенос
  комментария на отдельную строку.
- Общий UI: увеличены отступы, поля и скругления; глобальный размер компонентов
  Element Plus задан как `large`.
- `manifest.json`: добавлены host-разрешения для `localhost` и тестового
  инстанса `youtrack-test.sovcombank.ru`.

### Исправлено

- Мелкие правки стилей и обновление зависимостей (`package-lock.json`).
